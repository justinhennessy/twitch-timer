API Documentation

This documentation provides details about the API endpoints and how to use them.

1. Create Timer

POST /api/create

Creates a new timer and returns its UUID.

curl -X POST http://www.streamtimers.com/api/create

Response:

{
    "uuid": "your-uuid-here"
}

2. Get Timer

GET /api/timer/<uuid>

Gets the remaining time for the timer with the specified UUID.

curl http://www.streamtimers.com/api/timer/your-uuid-here

Response:

{
    "timer": remaining_time_in_seconds
}

3. Add Time

GET /api/add/<uuid>?t=<seconds>

Adds the specified number of seconds to the timer with the specified UUID. If t=infinite, sets the timer to infinite.

curl http://www.streamtimers.com/api/add/your-uuid-here?t=30

Response:

{
    "message": "Added 30 seconds",
    "timer": new_remaining_time_in_seconds
}

4. Reduce Time

GET /api/reduce/<uuid>

Reduces 30 seconds from the timer with the specified UUID.

curl http://www.streamtimers.com/api/reduce/your-uuid-here

Response:

{
    "message": "Reduced 30 seconds",
    "timer": new_remaining_time_in_seconds
}

5. Reset Timer

GET /api/reset/<uuid>

Resets the timer with the specified UUID.

curl http://www.streamtimers.com/api/reset/your-uuid-here

Response:

{
    "message": "Timer has been reset",
    "timer": new_remaining_time_in_seconds
}

6. Start Timer

GET /api/start/<uuid>

Starts the timer with the specified UUID.

curl http://www.streamtimers.com/api/start/your-uuid-here

Response:

{
    "message": "Timer started",
    "timer": new_remaining_time_in_seconds
}
