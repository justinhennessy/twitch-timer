API Documentation

This documentation provides details about the API endpoints and how to use them.

1. Get Timer

GET /api/timer/<uuid>

Gets the remaining time for the timer with the specified UUID.

Curl Example

curl https://streamtimers.com/api/timer/{{ uuid }}

Response:

{
    "timer": remaining_time_in_seconds
}

NightBOT Example

Command: !gettimer

$(eval const r=`$(urlfetch json https://streamtimers.com/api/timer/{{ uuid }})`; try { const j=JSON.parse(r); j.timer || 'Unexpected response'; } catch(e) { `Error parsing JSON. Raw response: ${r}`; })

2. Add Time

GET /api/add/<uuid>?t=<seconds>

Adds the specified number of seconds to the timer with the specified UUID. If t=infinite, sets the timer to infinite.

Explanation: The ?t=30 parameter specifies the number of seconds to add to the timer. You can change the value of 30 to any other number to add a different amount of time.

Curl Example

curl https://streamtimers.com/api/add/{{ uuid }}?t=30

Response:

{
    "message": "Added 30 seconds",
    "timer": new_remaining_time_in_seconds
}

NightBOT Example

Command: !addtime

$(eval const r=`$(urlfetch json https://streamtimers.com/api/add/{{ uuid }}?t=30)`; try { const j=JSON.parse(r); j.message || 'Unexpected response'; } catch(e) { `Error parsing JSON. Raw response: ${r}`; })

3. Reduce Time

GET /api/reduce/<uuid>

Reduces 30 seconds from the timer with the specified UUID.

Explanation: The ?t=30 parameter specifies the number of seconds to reduce from the timer. You can change the value of 30 to any other number to reduce a different amount of time.

Curl Example

curl https://streamtimers.com/api/reduce/{{ uuid }}?t=30

Response:

{
    "message": "Reduced 30 seconds",
    "timer": new_remaining_time_in_seconds
}
